CSS Box Model

The CSS box model consists of four parts: content area, padding, border, and margin.

The box-sizing property can be used to switch between the content-box and border-box models.

Content Box Model

This div uses the content box model. Its total width is 200px (content) + 20px (padding) + 10px (border) + 20px (margin) = 250px.

Border Box Model

This div uses the border box model. Its total width is 200px (including padding and border), and then add 20px (margin) = 220px.
